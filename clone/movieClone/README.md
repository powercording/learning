> 1. 애니메이션의 중복 동작으로 인하여 부자연스러운 연출이 발생하였습니다. 동작중에는 다른 애니메이션이 발생하지 않도록 상태를 변경하여 애니메이션의 중복동작을 방지하였습니다.
